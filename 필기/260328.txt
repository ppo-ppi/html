스타일을 입힐 요소를 찾는것

공통으로 넣는건 똑같이 넣기. 다르게 되는건 다르게 넣기.
    <div class="content-wrap cnt-wrap1">
        <div class="content cmt1">
            
        </div>
    </div>


> => 자식
+ => 형제

링크로 불러오기 => 외부에서
인라인 css => 각 태그 안에넣기 (지양하기) 우선순위가 가장 높음
내부 스타일 => 스타일태그

선택자 잡을때 틀릴때가 있으니 복붙하는게 좋음.
태그랑 클래스명을 같이쓰는게 우선순위가 높다. "명확도를 높힘"

header>div => 모든 자식 div를 잡는다.

공통을 잡지 않으면 스타일이 반복되게된다.

선택자를 나열할때는, 특정 선택자 아래에 모든 선택자가 들어간다.
헤더는 헤더끼리 모든 최 하위까지 나열~
        /* 컨텐츠 공통*/
        .content-wrap{}
        .content-wrap>.content{}
        .content-wrap>.content>header{}
        .content-wrap>.content>header>h2{}
        .content-wrap>.content>header>h4{}
        .content-wrap>.content>section{}


class가 두개 들어가있을때는 띄어쓰기 없이 나열한다.
