가로는 바뀌기때문에 20px이상은 숫자로 넣지않고 calc로 계산해서 넣는다. 20정도는 괜찮

아트탑 텍스트 패딩 왼쪽 계산하는 방법 -> TEXT => calc(100% / ) => width를 준 적 없기때문에 100%는 아티클이다.

display: flex는 width값이 없으면 원래의 가로 길이만큼만 차지한다.
👉 calc() 안의 %는 항상 부모 기준 퍼센트라고 보면 된다. => 내 자신의 width가 아니다!
부모에 width값이 설정되어있던지 말던지 아무튼 부모 기준이다.
전임자가 공통으로 쪼개놓은 선택자들을, 더 깊은 선택자를 작성해서 덮는다.
flex하면 높이가 큰쪽 높이로 잡힌다.

공통화 = 공통화된게 많은것을 넣고, 특별한애는 따로 집어서 넣어준다.

텍스트 => 줄수 제한하는방법 line-height와 height 그리고 overflow-hidden 써야한다.
height: 
버튼은 라인하이트도, 하이트도 되지만 a태그로 대체될대를 대비해서 lh를 준다.

부모보다 너비가 작으면 width값을 줘야한다.
wrap보다 안에있는 grid가 작아야 가운데 정렬이 가능하다. 그래서 width값이 있을때만 가능하다.
