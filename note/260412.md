핵심 원인 1: flex는 “자식 크기에 의존”

flex 컨테이너에서 자식들은 기본적으로:

flex: 0 1 auto;

👉 즉,

내용(content) 기준으로 크기 잡음
❗ 문제 상황
1️⃣ 원래 상태
.image { height: 320px; }
.texts { padding 있음 }

👉 둘 다 크기가 있어서 정상 표시

2️⃣ .texts { display: none }

👉 이 순간:

.texts 완전히 사라짐
flex는 .image만 보고 레이아웃 계산
🔴 그런데 왜 이미지도 안 보였냐?

👉 여기서 중요한 포인트:

.image {
    background-image: url(...);
    height: 320px;
}

겉보기엔 높이가 있지만…

💣 flex 환경에서는

👉 width가 없으면 0이 될 수 있음

🔥 핵심 원인 2: flex 아이템의 width

flex에서는:

width 지정 없으면
내용 기반으로 줄어듦

근데 .image는?

background-image
+ 텍스트 없음 (color: transparent)

👉 실제 콘텐츠가 없음 → width 계산 = 0 가능

💥 결과
상태	결과
texts 있음	이미지 + 텍스트 둘 다 보임
texts 없음	이미지 width 0 → 안 보임
🔧 해결 방법
✅ 방법 1 (가장 확실)
article {
    display: flex;
}

.image {
    width: 100%;
}

또는

.image {
    flex: 1;
}
✅ 방법 2
article {
    display: flex;
}

.image {
    min-width: 200px;
}
✅ 방법 3 (추천 구조)
article {
    display: flex;
}

.image {
    flex: 1;
}

.texts {
    flex: 1;
}
